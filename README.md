# Movie
### 프로젝트:
-  영화 예매 웹 페이지 제작

### 목표:
-  사용자 정보 입력 후 영화 목록 제공 및 예매 정보를 저장하는 시스템 구현

### 주요 업무:
 - 사용자 정보 입력 후 영화 목록 제공 및 예매 정보 저장 기능 구현
 - SQL을 활용하여 예매 정보가 DB에 저장되고 수정되는 과정 구현
 - 데이터베이스 흐름 및 구조적 설계 경험

### 사용 기술:
- MySQL
- PHP
